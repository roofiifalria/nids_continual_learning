# Continual Learning NIDS (NF_UQ_NIDS_V2)

This project implements a **continual learning framework** for Network Intrusion Detection Systems (NIDS) using **Reinforcement Learning (RL)** with **PPO**, a **Conditional GAN (CGAN)**, and an **incremental MLP classifier**.

The goal is to automate fine-tuning of the MLP classifier on new or unseen traffic patterns, generated by the CGAN or retrieved from a replay buffer, while preventing catastrophic forgetting.

---

## 🧩 Components

- **`preprocessing.py`** — cleans and scales the NF_UQ_NIDS_V2 dataset.
- **`dataloader.py`** — loads preprocessed train/test splits.
- **`mlp_classifier.py`** — PyTorch-based MLP classifier with incremental updates.
- **`cgan_torch.py`** — PyTorch Conditional GAN for generating class-conditioned samples.
- **`replay_buffer.py`** — stores seen samples to avoid forgetting.
- **`envs.py`** — defines RL environments (`real`, `gan`, `hybrid`).
- **`ppo_agent.py`, `ppo_runner.py`** — PPO agent implementation and training loop.
- **`main.py`** — orchestrates the full continual-learning pipeline.

## 📂 Dataset Preparation

Download or locate the **NF_UQ_NIDS_V2** dataset.
Place the CSVs under a folder, e.g. `datasets/NF_UQ_NIDS_V2/`.

Then preprocess them:
```bash
python src/data/preprocessing.py datasets/NF-UQ-NIDS-V2.csv --label_col Attack --save_dir preprocessed --parquet --chunksize 200000
```

This creates:
```
preprocessed/
├── train_df.csv
├── test_df.csv
└── label_dict.json
```

---

## 🚀 Running the Full Pipeline

Run `main.py` to launch end-to-end continual learning training.

### Example:
```bash
python src/main.py \
  --data_root . \
  --target_class DDoS \
  --seen_cap_total 200000 \
  --seen_cap_per_class 30000 \
  --unseen_cap 80000 \
  --test_cap_per_class 5000 \
  --ppo_steps 20000 \
  --gan_epochs 10 \
  --mlp_epochs 2
```

### Modes
- `--env real` → uses only real samples.
- `--env gan` → uses only CGAN-generated samples.
- `--env hybrid` → uses both CGAN and replay buffer (recommended).

## 📈 Customization
- **Reward**: ΔF1-score (default) or ΔAccuracy (`--delta_acc`).
- **GAN control**: adjust `--gan_noise`, `--gan_hidden`, `--gan_glayers`, etc.
- **Replay buffer**: set `--replay_cap` and initial seeding size with `--seed_replay`.
- **PPO settings**: learning rates, clip, gamma, lambda, etc.

## 🧪 Example Workflow

```bash
# Step 1: preprocess
python preprocessing.py datasets/NF_UQ_NIDS_V2

# Step 2: train end-to-end continual model
python src/main.py datasets/NF_UQ_NIDS_V2 --env hybrid

# Step 3: evaluate classifier
python -c "from mlp_classifier import MLPClassifierTorch; clf=MLPClassifierTorch.load('outputs/.../checkpoints/mlp_final.pt')"
```
